EC2 Feedback Form Project (Auto Scaling)

Built a feedback form web application using HTML & CSS.
Deployed on EC2 instances using a Launch Template.
Configured Auto Scaling Group (Min:1, Max:3) for dynamic scaling.
Implemented Application Load Balancer (ALB) to distribute traffic.
Instance ID displayed on the page to demonstrate load balancing.
Feedback data stored securely in Amazon RDS (MySQL).
CloudWatch monitors CPU metrics and triggers Auto Scaling.
Auto Scaling replaces unhealthy EC2 instances automatically.
High availability achieved across multiple Availability Zones.
Access the live application: feedback-loadbalancer-370182017.ap-south-1.elb.amazonaws.com
